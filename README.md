# Lahore Boys Hostels

A comprehensive hostel directory and management platform for boys hostels in Lahore, Pakistan.

## Features

- 🏢 12 Verified Hostels Directory
- 🔍 Advanced Search & Filtering
- 💰 Transparent Pricing
- 📊 Hostel Management Portal
- 👥 Multi-role System (Manager, Admin, Hostelite, Staff, Finance)
- 💾 Local Storage (No server required)
- 🌙 Dark Mode Support
- 📱 Responsive Design

## Quick Start

### Local Development
```bash
# Open the file directly in your browser
open index.html

# Or run a local server
npx http-server
```

### Deployment to Vercel

1. **Create a GitHub repository**
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Lahore Boys Hostels"
   git branch -M main
   git remote add origin https://github.com/awaisrehman936/lahore-boys-hostels.git
   git push -u origin main
   ```

2. **Deploy to Vercel**
   - Go to [vercel.com](https://vercel.com)
   - Sign up with your GitHub account (awaisrehman936@gmail.com)
   - Click "New Project"
   - Import your GitHub repository
   - Vercel will auto-detect the configuration and deploy
   - Your site will be live at `https://lahore-boys-hostels.vercel.app`

## Features Overview

### Public Directory (Home Page)
- Browse 12 verified hostels
- Filter by category, price, amenities
- Search by area or hostel name
- View hostel details, amenities, pricing
- Submit applications directly

### Portal System
Six portal roles available:

1. **Manager Portal** - Manage tenants, rooms, payments, maintenance
2. **Admin Portal** - Oversee all hostels, manage groups, analytics
3. **Hostelite Portal** - Personal dashboard, room details, payment history
4. **Staff Portal** - Task management, attendance tracking
5. **Finance Portal** - Ledger, expense tracking, payments
6. **Regular User Portal** - Application tracking

### Key Features
- ✅ Tenant Management with move-in/out tracking
- ✅ Room allocation and status tracking
- ✅ Payment collection and expense management
- ✅ Maintenance issue tracking
- ✅ Communication logs
- ✅ Health score calculation
- ✅ Analytics and reporting
- ✅ Late fee management
- ✅ Settlement workflow
- ✅ Visitor tracking
- ✅ Attendance management
- ✅ Document management
